# STEREO (3D WebGL Virtual Game Engine)

STEREO は、WebGL (Three.js 等) を最大限に活用した 3D 専用仮想ゲームエンジンです。
MONORAL (2D 仮想ゲームエンジン) の親ホスト環境として機能し、3Dゲームセンター空間などの多重入れ子エミュレーションを実現します。

## 主な役割
* **3D 仮想世界の構築**: Voxel やローポリゴンのゲームセンター/室内ロビーなどを構築する。
* **EngineScript 3D 拡張の解釈**: C# / ENGINEv4 互換の3D空間占有 (`gps`), データベース (`db`), タスク並列 (`task`) などの API を WebGL 上でシミュレート・翻訳実行する。
* **MONORAL 筐体エミュレーション**: 空間内のアーケード筐体のモニターに対し、子プロセスである MONORAL 2D エミュレータを動的マッピングする。
